# Torus Edison マニュアル

英語版: [Manual.md](Manual.md)

## 概要

`Torus Edison` は、再利用しやすいゲームオーディオ用プロジェクトデータを Unity Editor 上で作成、保存、試聴するためのエディタ拡張です。

現時点で利用できる主な機能:

- `Tools/Torus Edison/Open Editor` からの起動
- 新規プロジェクトの作成
- `.gats.json` プロジェクトファイルの保存と読み込み
- オフライン音声バッファのレンダリング
- Unity Editor 上での `Play / Pause / Stop / Rewind / Loop` プレビュー再生
- サンプルプロジェクトを使った確認

このマニュアルは現行実装を前提にしています。仕様書上の全機能が揃った最終版マニュアルではありません。

## 導入

1. Unity プロジェクト内の `Packages/com.sunmax.trusedison` にパッケージを配置します。
2. Unity プロジェクトを開きます。
3. パッケージの読み込みが完了するまで待ちます。

## 起動方法

次のメニューから起動します。

- `Tools/Torus Edison/Open Editor`

## 保存と読み込み

- `New`
  新しいプロジェクトを作成します。
- `Open`
  `.gats.json` プロジェクトファイルを読み込みます。形式が不正な場合は読み込みを拒否します。
- `Save`
  現在のプロジェクトを保存します。
- `Save As`
  別名保存します。保存時は `.gats.json` 拡張子に正規化されます。

標準のセッション形式は `.gats.json` です。

## プレビュー再生

- `Render Preview`
  現在のプロジェクト内容からプレビュー用の音声バッファを生成します。
- `Play`
  プレビュー再生を開始します。
- `Pause`
  現在位置を保持したまま一時停止します。
- `Stop`
  再生を停止して先頭に戻します。
- `Rewind`
  音声を再生成せずにカーソルだけ先頭に戻します。
- `Loop`
  `Total Bars` を基準にループ再生します。

## 設定ファイル

現在の基盤実装では、以下の設定ファイルを利用します。

- 共通設定: `%LocalAppData%/GameAudioTool/config.json`
- プロジェクト設定: `ProjectSettings/GameAudioToolSettings.json`

プロジェクト設定は、サンプルレート、チャンネルモード、出力先ディレクトリの既定値について共通設定より優先されます。設定 JSON が壊れている場合は、ビルトイン既定値にフォールバックします。
